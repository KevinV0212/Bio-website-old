Event Texter

Event texter was a submission to the KnightsHacks 2021 Hackathon and won the "Most Creative Use of Twillio" prize. It is a simple java-based web application created using the Vaadin platform. EventTexter enables users to coordinate events with recipients in a list via text through the Twillio API.

Contribution

My role in this project was researching the Twillio API and connecting it to EventTexter.
